Image
The Image component is used to display images. It can be responsive or of a fixed size and shape.
Responsive Images
Images can be made responsive with 'img-responsive' class. Use this if you want to scale image with the parent's width. 
                
  <img src="..." class="img-responsive" alt="..." />
Square Images
The squared image, crops the image into a square. You can make square images in one of the 3 sizes. Using the class 'img-sq-sm | img-sq-md | img-sq-lg' to create small image of 150x150, medium of 200x200 or large of 250x250. 
                
  <img src="..." class="img-sq-sm" alt="..." />
  <img src="..." class="img-sq-md" alt="..." />
  <img src="..." class="img-sq-lg" alt="..." />
Round Images
Round images can be created using 'img-round' class. For best results use it with the 'Square Images' classes. 
                
  <img src="..." class="img-sq-sm img-round" alt="..." />
  <img src="..." class="img-sq-md img-round" alt="..." />
  <img src="..." class="img-sq-lg img-round" alt="..." />
