Slider
Sliders allow users to make selections from a range of values. Two types of sliders are available, basic slider and steps slider.
Basic Slider
To create a basic slider, use 'slider' class on <input type="range"> element. 
                
    <label for="demoSlider1">Example range</label>
    <input type="range" class="slider" id="demoSlider1" />
Steps Slider
Range inputs have implicit values for min and maxâ€”0 and 100, respectively. New values can be specified for those using the min and max attributes. By default, range inputs "snap" to integer values. To change this, we can specify a step value. In the example below, we double the number of steps by using step="0.5". 
                
  <label for="demoSlider2">Example range</label>
  <input type="range" class="slider" min="0" max="5" step="0.5" id="demoSlider2" />
