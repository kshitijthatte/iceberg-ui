Navigation
Navigation component has a navbar, that can be used to display important information about the page and carry the action items related to the current page or can be used to move to a diffrent page.
Standard Navigation
To create a standard nav bar, use 'nav' class on an element and use 'nav-section' inside the 'nav' to create diffrent section. To create a right aligned section use the class, 'nav-section-right' along with 'nav-section' class.
View Demo 
                
  <nav class="nav">
    <div class="nav-section">
      <button class="material-icons nav-icon">menu</button>
      <span class="nav-title">Brand Name</span>
    </div>
    <div class="nav-section nav-section-right">
      <button class="material-icons nav-icon">file_download</button>
      <button class="material-icons nav-icon">print</button>
      <button class="material-icons nav-icon">bookmark</button>
    </div>
  </nav>
Fixed Navigation
To create a fixed navbar use class 'nav-fixed' along with 'nav' class. Also to avoid overlapping use 'nav-fixed-adjust' adjust class on the element next to 'nav-fixed' navbar.
View Demo 
                  
    <nav class="nav nav-fixed">
      <div class="nav-section">
        <button class="material-icons nav-icon">menu</button>
        <span class="nav-title">Brand Name</span>
      </div>
      <div class="nav-section nav-section-right">
        <button class="material-icons nav-icon">file_download</button>
        <button class="material-icons nav-icon">print</button>
        <button class="material-icons nav-icon">bookmark</button>
      </div>
    </nav>

    <div class="nav-fixed-adjust">
      ...
    </div>
