Card
Cards are containers that can display content and actions on a single topic. They should be easy to scan for relevant and actionable information.
Basic Card
Create a basic card with img and description by using 'card' class on a div and giving 'card-img' & 'card-description' class to it's child image and div element.

  <div class="card">
    <img class="card-img" src="https://picsum.photos/300/200" />
    <div class="card-description">
      Lorem ipsum dolor sit amet consectetur adipisicing elit. Quae, nam!
    <div>
  <div>

Card with Action Buttons
Card with Action Buttons can be created by using the 'card-actions' class on a div, inside the 'card' component. To position actions in the right end of the card use 'card-actions-right' class on a div, inside the 'card-actions'.

Cards with Text Overlay
Cards with text overlay can be created by using the 'card-overlay' class along with the 'card' class. Then wrap the content that you want to overlay in a div with class 'overlay-content'. This div must be next sibling to 'card-img' element.

by Michael Scarn

Horizontal Card
Horizontal Card can be created using the 'card-horizontal' class. on a div, inside the 'card'. With that use the 'card-content' class to put the card content horizontally along the image.
Our Changing Planet
by Michael Scarn
                
  <div class="card">
    <div class="card-horizontal">
      <img class="card-img" src="https://picsum.photos/300/200" />
      <div class="card-content">
        <div class="card-title">Our Changing Planet</div>
        <div class="card-subtitle">by Michael Scarn</div>
      </div>
    </div>
    <div class="card-actions">
      <button class="btn btn-sm btn-transparent">Read</button>

        <button class="btn card-icon material-icons">favorite_border</button>
