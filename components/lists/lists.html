Lists
A list can be used for displaying a series of content. The content can consist of multiple elements of varying type and size.
Spaced Lists
To create a normal spaced list, use class 'list-group' on a 'ul' tag and then inside that use class 'list-item' for children list items.
  • An item
  • A second item
  • A third item
  • A fourth item
  • And a fifth one
                
  <ul class="list-group">
    <li class="list-item">An item</li>
    <li class="list-item">A second item</li>
    <li class="list-item">A third item</li>
    <li class="list-item">A fourth item</li>
    <li class="list-item">And a fifth one</li>
  </ul>
Stacked Lists
To create Stacked Lists, inside 'list-group' element use class 'list-item-stacked' oon list items.
  • An item
  • A second item
  • A third item
  • A fourth item
  • And a fifth one
                  
  <ul class="list-group">
    <li class="list-item-stacked">An item</li>
    <li class="list-item-stacked">A second item</li>
    <li class="list-item-stacked">A third item</li>
    <li class="list-item-stacked">A fourth item</li>
    <li class="list-item-stacked">And a fifth one</li>
  </ul>
