Avatar
The Avatar component is used to represent a user, and displays the profile picture, initials or an icon.
Circular avatars
Circular avatars can be created by using the 'avatar' class inside the img tag. 
                
<img class="avatar" src="/assets/avatar/1.jfif" alt="" />
<img class="avatar" src="/assets/avatar/2.jfif" alt="" />
<img class="avatar" src="/assets/avatar/3.jfif" alt="" />
Rounded avatars
Rounded avatars can be created by using the 'avatar-rounded' class along with the 'avatar' class inside the img tag. 
                
<img class="avatar avatar-rounded" src="/assets/avatar/1.jfif" alt="" />
<img class="avatar avatar-rounded" src="/assets/avatar/2.jfif" alt="" />
<img class="avatar avatar-rounded" src="/assets/avatar/3.jfif" alt="" />
Avatar Sizes
The Avatar component comes in 5 sizes. It can be changed by adding 'avatar-xs | avatar-sm | avatar-md | avatar-lg | avatar-xl' class to the existing 'avatar' class. 
                
<img class="avatar avatar-xs" src="/assets/avatar/1.jfif" alt="" />
<img class="avatar avatar-sm" src="/assets/avatar/1.jfif" alt="" />
<img class="avatar avatar-md" src="/assets/avatar/1.jfif" alt="" />
<img class="avatar avatar-lg" src="/assets/avatar/1.jfif" alt="" />
<img class="avatar avatar-xl" src="/assets/avatar/1.jfif" alt="" />
With text
The Avatar Component with text can be created using the 'avatar-container' class and then placing 'avatar' inside it. Using the 'avatar-title' & 'avatar-subtitle' class the following style can be achieved.
Michael Scarn
View Profile
                
<div class="avatar-container">
  <img class="avatar" src="/assets/avatar/1.jfif" alt="Michael Scarn" />
  <div class="avatar-details">
    <div class="avatar-title">Michael Scarn</div>
    <div class="avatar-subtitle">View Profile</div>
  </div>
</div>
