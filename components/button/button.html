Button
The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.
Button variants
To create a button use a 'btn' class along with the prefered variant class from 'btn-primary | btn-secondary | btn-success | btn-error'. 
                
  <button class="btn btn-primary">Primary</button>
  <button class="btn btn-secondary">Secondary</button>
  <button class="btn btn-success">Success</button>
  <button class="btn btn-error">Error</button>
Icon Button
Icon Buttons can be created, by using a icon img as a child to parent button. Use the 'btn' class followed by 'btn-icon'. 
                
  <button class="btn btn-primary btn-icon">
    <img src="/assets/icons/shopping-cart_icon.svg" alt="cart" />
  </button>
  <button class="btn btn-primary btn-icon">
    <img src="/assets/icons/shopping-cart_icon.svg" alt="cart" />
    Cart
  </button>
Link Button
To create a link button, no especial classes are needed. Just use an a tag inside the 'btn' component. 
                
  <button class="btn btn-primary">
    <a href="https://iceberg-ui.netlify.app/" target="_"> iceberg </a>
  </button>
  <button class="btn btn-primary btn-icon">
    <img src="facebook_icon.svg" alt="facebook_icon" />
    <a href="https://www.facebook.com/">facebook</a>
  </button>
Round Button
To create a Round button add the 'btn-round' class to the 'btn' component. 
                
  <button class="btn btn-primary btn-round">Primary</button>
  <button class="btn btn-secondary btn-round">Secondary</button>
  <button class="btn btn-success btn-round">Success</button>
  <button class="btn btn-error btn-round">Error</button>
Outline Button
In need of a button, but not the hefty background colors they bring? Replace '.btn-variant' with '.outline-variant'. 
                
  <button class="btn outline-primary">Primary</button>
  <button class="btn outline-secondary">Secondary</button>
  <button class="btn outline-success">Success</button>
  <button class="btn outline-error">Error</button>
Button Sizes
The Button component comes in 3 sizes. It can be changed by adding 'btn-sm | btn-md | btn-lg' class to the existing 'btn' class. 
                
  <button class="btn btn-primary btn-sm">Primary</button>
  <button class="btn btn-primary btn-md">Primary</button>
  <button class="btn btn-primary btn-lg">Primary</button>
Transparent Button
To make a Transparent Button add the 'btn-transparent' class to the 'btn' component. 
                
  <button class="btn btn-transparent btn-sm">Primary</button>
  <button class="btn btn-transparent btn-md">Primary</button>
  <button class="btn btn-transparent btn-lg">Primary</button>
