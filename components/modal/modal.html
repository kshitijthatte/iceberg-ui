Basic Modal
To create a basic modal, use 'modal' class on a div. Inside the 'modal' use 'modal-title' class for modal title and wrap the modal action buttons inside the 'modal-actions' div.

  <div class="modal">
    <div class="modal-title">Modal title</div>
    <div>Modal body text goes here.</div>
    <div class="modal-actions">
      <button class="btn btn-transparent btn-sm">Close</button>
      <button class="btn btn-transparent btn-sm">Save</button>
    </div>
  </div>
