Alert
The Alert component is used to convey important information to the user. These default types come in 4 variations: primary, success, warning, and error.
Simple Alert
Simple Alert can be created using alert class, followed by the variant name.
A simple primary alert—check it out!
A simple success alert—check it out!
A simple warning alert—check it out!
A simple error alert—check it out! 
                
  <div class="alert alert-primary">
    A simple primary alert—check it out!
  </div>
  <div class="alert alert-success">
    A simple success alert—check it out!
  </div>
  <div class="alert alert-warning">
    A simple warning alert—check it out!
  </div>
  <div class="alert alert-error">
    A simple error alert—check it out!
  </div>
Outline Alert
Outline alert can be created by using 'alert' class, followed by 'outline-variant'.
An outline primary alert—check it out!
An outline success alert—check it out!
An outline warning alert—check it out!
An outline error alert—check it out! 
                
  <div class="alert outline-primary">
    An outline primary alert—check it out!
  </div>
  <div class="alert outline-success">
    An outline success alert—check it out!
  </div>
  <div class="alert outline-warning">
    An outline warning alert—check it out!
  </div>
  <div class="alert outline-error">
    An outline error alert—check it out!
  </div>
Alert with accent border
Alert with accent border can only be created with Simple Alerts.
Utility classes 'alert-b-t | alert-b-r | alert-b-b | alert-b-l' can be used to give border on top | right | bottom | left.
A simple primary alert with accent border—check it out!
A simple success alert with accent border—check it out!
A simple warning alert with accent border—check it out!
A simple error alert with accent border—check it out! 
                
  <div class="alert alert-primary alert-b-l">
    A simple primary alert with accent border—check it out!
  </div>
  <div class="alert alert-success alert-b-t">
    A simple success alert with accent border—check it out!
  </div>
  <div class="alert alert-warning alert-b-b">
    A simple warning alert with accent border—check it out!
  </div>
  <div class="alert alert-error alert-b-r">
    A simple error alert with accent border—check it out!
  </div>
